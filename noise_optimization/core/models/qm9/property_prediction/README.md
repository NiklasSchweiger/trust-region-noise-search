# QM9 property prediction (EGNN classifiers)

This directory contains the **property prediction** code and pre-trained checkpoints used by the multi-property QM9 reward. The implementation and checkpoints are taken from:

- **Guided Flow Matching with Optimal Control**
  - GitHub: https://github.com/WangLuran/Guided-Flow-Matching-with-Optimal-Control
  - arXiv: https://arxiv.org/abs/2410.18070

Pre-trained classifiers for all six QM9 properties (`alpha`, `gap`, `homo`, `lumo`, `mu`, `Cv`) are included under `outputs/`. No additional setup is required.
